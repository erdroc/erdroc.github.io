Bullfrog

Autonomous Unmanned Ackermann-Steered Ground Vehicle "Bullfrog", 03/09/2023

This project is similar to the ladybug, but using Ackermann Steering. The goal is to build a robot that can drive quickly and handle a few bumps on the ground. It is in early development and is using an old toy remote control car chassis. This page serves as my documentation for this project and I plan to keep updating it as things progress.
